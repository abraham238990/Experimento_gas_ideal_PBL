# Experimento_gas_ideal_PBL
Este repositório contém os programas usados para o experimento do Gás ideal de Práticas Básicas de Laboratório.

## Instalação

Para executar o projeto localmente em seu computador, você precisará baixar este repositório. Para isso, você pode:

- Clonar o repositório; ou  
- Baixar o arquivo `.zip`.

Após isso, instale as bibliotecas necessárias executando o seguinte comando no terminal:

```bash
pip install matplotlib pandas numpy scipy scikit-learn
```
